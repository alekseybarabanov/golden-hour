# kits/ — отдельные дистрибутивы

Папка **не является частью** агента «Золотой час» (`SOUL.md`, `scripts/`, `skills/` в корне).

Каждый подкаталог — самостоятельный кит со своей установкой и документацией. Агент golden-hour их не загружает и не исполняет.

| Кит | Назначение |
|---|---|
| `notes-bot-kit-v1/` | Telegram-бот конспектов и заметок (Python) |

Установка — см. `INSTALL.md` внутри соответствующего кита.
