# Параметры / исследование — вводный блок

**TL;DR:** Три базовых приёма для квадратных уравнений с параметром — дискриминант (D ≥ 0 для существования корней), теорема Виета (сумма и произведение корней), разбор вырожденных случаев (когда коэффициент при x² обнуляется). На ФМЛ №239 — 4–5 задач такого типа из ~20.

## Суть

### 1. Дискриминант
- Уравнение `ax² + bx + c = 0` (a ≠ 0) имеет корни ⟺ D = b² − 4ac ≥ 0.
- D = 0 → один корень (кратный), D > 0 → два различных.
- **Ловушка:** перед применением формулы проверить, что a ≠ 0. Если a = 0, уравнение становится линейным (или вырожденным).

### 2. Теорема Виета (для корней x₁, x₂)
- x₁ + x₂ = −b/a
- x₁ · x₂ = c/a
- **Применение:** знаки корней (по знаку произведения), соотношения (x₁ = 2x₂ → x₁/x₂ = 2 → x₁·x₂ = 2x₂² → ...), расположение на числовой оси.

### 3. Расположение корней
- Оба положительны: D ≥ 0, x₁ + x₂ > 0, x₁ · x₂ > 0.
- Оба отрицательны: D ≥ 0, x₁ + x₂ < 0, x₁ · x₂ > 0.
- Разных знаков: x₁ · x₂ < 0 (D ≥ 0 автоматически).
- Оба ≥ k: замена y = x − k, применить к новому уравнению.

## Формулы

- D = b² − 4ac
- x = (−b ± √D) / (2a)
- Сумма: −b/a, Произведение: c/a

## Задачи

### Задача 1 — на дискриминант (базовая)
**При каких a уравнение `x² − 2ax + 3a = 0` имеет ровно один корень?**

- D = 4a² − 12a = 4a(a − 3) = 0 → a = 0 или a = 3.
- **a = 0:** уравнение 0 = 0, бесконечно решений → **не подходит** (типичная ловушка: проверить, что не выродилось).
- **a = 3:** x² − 6x + 9 = 0 → (x − 3)² = 0 → x = 3 — ровно один корень ✓.
- **Ответ:** a = 3.

### Задача 2 — на теорему Виета (средняя)
**При каких a оба корня уравнения `x² + (a − 1)x + a² − 6 = 0` положительны?**

Условия:
1) **D ≥ 0:** (a − 1)² − 4(a² − 6) ≥ 0
   → a² − 2a + 1 − 4a² + 24 ≥ 0
   → −3a² − 2a + 25 ≥ 0
   → 3a² + 2a − 25 ≤ 0
   D_удвоенного = 4 + 300 = 304
   a = (−2 ± √304) / 6 = (−1 ± 2√19) / 3
   ≈ a ∈ [−3.24, 2.57]
2) **Сумма > 0:** −(a − 1) > 0 → a < 1.
3) **Произведение > 0:** a² − 6 > 0 → |a| > √6 ≈ 2.45 → a < −√6 или a > √6.

Пересечение: a ∈ [−3.24, −√6) ∪ (√6, 1).
- **√6 ≈ 2.449**
- **Ответ:** a ∈ [−(1 + 2√19)/3, −√6) ∪ (√6, 1).
