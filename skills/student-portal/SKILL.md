# student-portal — личный кабинет (Wi‑Fi)

Минимальный веб-интерфейс для **одного** пользователя: план на сегодня и чат с ботом. Работает с телефона/планшета в **той же Wi‑Fi сети**, что и ПК с OpenClaw.

## Когда

- `setup_status: complete`
- Триггеры: `/web`, «личный кабинет», «веб-план», «ссылка на план в браузере»

## Действие агента

```bash
node scripts/student-portal.mjs --user <user_key>
```

1. Взять `portal_url` из JSON.
2. Отправить пользователю коротко:

> 🌅 **Личный кабинет** (тот же Wi‑Fi):  
> <portal_url>  
> План на сегодня и чат — открой в браузере телефона.

3. **Не** показывать: `user_key`, telegram id, токен, пути к файлам.

## Запуск (владелец)

```powershell
cd dashboard
.\start_student_portal.ps1
```

Порт **18791**, слушает `0.0.0.0` (LAN).

## Безопасность

- Ссылка = секрет; не пересылать чужим.
- Доступ только в локальной сети (не через интернет).
- Gateway остаётся на loopback; чат идёт через прокси портала.
