# help-menu — что умеет бот

Единая точка, где пользователь узнаёт обо **всех** функциях.

## Триггеры
- «что умеешь», «помощь», «меню», «функции», «/help», «?»
- Авто: после завершения настройки (`onboarding` шаг 7)
- Контекстно: «а ты можешь …?»

## Поведение по фазе
- **`setup_status ≠ complete`** — коротко: «Сейчас настройка. После неё: план, задачи, команда, таймер, напоминания, статистика. Допройдём?»
- **`setup_status: complete`** — полное меню ниже.

## Полное меню (после настройки)

```
🌅 Вот что я умею, <name>. Просто напиши словами — пойму.

📅 ПЛАН
• «составь план» — макро-план до дедлайна
• «спланируй день» — сбалансированный день
• «карточки плана» / «/cards» — план картинками
• любая таблица / расписание — **сразу картинкой** (я не шлю таблицы текстом, не спрашиваю)
• «неделя 3» — задание на неделю
• «делай каждый день …» — повторяющиеся дела
• «что повторить» — слабые темы (1→3→7→14→30)
• «история по теме X» — граф событий

✅ ЗАДАЧИ
• «список задач» / «добавь» / «закрой»
• «разбей задачу …» — шаги <15 мин
• «что горит» / «прогресс»
• «… важнее» / «… тяжелее» — приоритет и сложность

👥 КОМАНДА
• «создай команду» / «мои команды»
• «пригласи @…» / «вступить по коду»
• «беру» / «сдал» / «принято»

⏱ ФОКУС И ДИСЦИПЛИНА
• «начни помодоро» / «поработаем 25/5»
• «давай час над X» — фокус-сессия
• «чек-ин» / «итог дня»
• утренний бриф, пинги, вечерний чек-ин («жёстче»/«мягче»)
• «не успел / провалил» — разбор без самобичевания

📚 МАТЕРИАЛЫ
• «дай материалы» / «подборка на сегодня» — сначала локальная библиотека, потом поиск

📊 СТАТИСТИКА
• «статистика за неделю/месяц» — прогресс и задачи

🔗 КАЛЕНДАРЬ
• «подключи календарь» / «синхронизируй» — Google Calendar

⚙️ НАСТРОЙКИ
• «настроить заново» — через меню при входе (с подтверждением)
• «жёстче»/«мягче» — напоминания
• «сделай день легче» — темп нагрузки
• `/web` — личный кабинет в браузере (план дня + чат, тот же Wi‑Fi)

Скажи, с чего начнём?
```

## Правила
- Только доступное в текущей фазе.
- **О себе — мужской род** («готов», «сделал»), не женский.
- **Не показывать** имена скриптов, путей, `user_key` — только человеческие команды.
- Тон: короткий, 🌅 в начале ключевых сообщений.
